## 📌 Proyecto destacado del bootcamp
### 🧠 Sistema de Deméritos para Estudiantes (Automatizado con n8n)

##### Desarrollé un sistema automatizado para gestionar deméritos estudiantiles utilizando n8n. Este proyecto permite registrar, procesar y notificar incidencias disciplinarias de manera eficiente.

##### 🔧 Funcionalidades principales:
* Registro automático de faltas mediante formularios
* Clasificación de deméritos según gravedad
* Notificaciones automáticas a estudiantes y encargados
* Integración con hojas de cálculo para almacenamiento de datos
* Generación de reportes periódicos

##### ⚙️ Tecnologías utilizadas:
* n8n (automatización de flujos)
* Webhooks
* Google Sheets (o base de datos)
* Correo electrónico (SMTP / Gmail API)

##### 🎯 Objetivo del proyecto:
Optimizar el proceso manual de registro de incidencias en instituciones educativas, reduciendo errores y mejorando la comunicación entre docentes, estudiantes y padres.
